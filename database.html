User¶

class database.User(email, first_name=None, birthday=None, gender=None, major=None, bio=None, password=None, profile_photo=None, verified=False, deactivated=False)[source]¶

This class models a user within the application, storing various attributes such as email, password, personal information, and account status.

Parameters:

  • user_id (int) – The id of the user.

  • email (str) – The email address of the user.

  • first_name (str, optional) – The first name of the user, defaults to None.

  • birthday (datetime.date, optional) – The birthday of the user, defaults to None.

  • gender (str, optional) – The gender of the user, defaults to None.

  • major (str, optional) – The major of the user, defaults to None.

  • bio (str, optional) – The bio of the user, defaults to None.

  • password (str, optional) – The password of the user, defaults to None.

  • profile_photo (str, optional) – The file name of the user’s profile photo, defaults to None.

  • verified (bool, optional) – Indicates whether the user’s TAMU email is verified, defaults to False.

  • deactivated (bool, optional) – Indicates whether the user’s account is deactivated, defaults to False.
